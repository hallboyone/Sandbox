\doxysubsection{Detailed Description}
Abstracts the usage of command line flags. Basic workflow is to construct an instance using the command line args (argc and argv) and then check status of flags using provided function calls (is\+Set\+With\+Para -\/$>$ check if flag is set and has a parameter of the given type).
